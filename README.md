
# Psych Mental Health Documentation Auditor

Created by Janyce Viero

## Project Overview

This is a behavioral health version of the AI Medical Documentation Auditor portfolio project.

It is designed for counseling, psychology, therapy, and mental health documentation review.

## What It Reviews

- Client reason for visit
- Session type or service provided
- Symptoms or functional status
- Intervention or modality used
- Client response and participation
- Progress toward treatment goals
- Risk assessment documentation
- Safety plan or crisis instructions
- Treatment plan and follow-up
- Behavioral health privacy sensitivity flags

## Important Disclaimer

This app is for educational HIM/AI portfolio use only. It does not provide final clinical, legal, compliance, billing, or coding advice.

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## How to Deploy

Upload these files to a GitHub repository and deploy through Streamlit Community Cloud.

Main file path:

```text
streamlit_app.py
```

## Portfolio Description

Developed an AI-assisted behavioral health documentation review app using Python and Streamlit. The tool evaluates counseling and psychology notes for documentation completeness, treatment plan support, risk assessment, privacy sensitivity, and clinician query opportunities.
